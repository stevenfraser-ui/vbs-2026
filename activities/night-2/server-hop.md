Split group into two teams: Alpha and Bravo
Both teams have a poster board with a grid of connected servers. Each server has a designation on it (e.g. A3, B5, etc...).
Both teams also have a stack of "pattern cards." Each pattern card has a designation on the back that relates to the designation on each server. The other side has some kind of pattern game. Some games are simpler for younger kids, some are harder for older kids.
But each team only has half of the cards.
They are instructed to start at the server designated A1. Only one of the teams has the A1 pattern card.
The team with the A1 card, figures out the pattern on the card to know how to move from the A1 server.
