# VBS 2026 Teaser Trailer Script (Concept 3: Recruit Journey)

Duration: ~30 seconds  
Style: AI-generated cinematic animation  
Tone: adventurous, inspiring, cinematic suspense

## Runway Gen-4.5 Setup

Use these settings for all five clips unless a scene says otherwise.

- Model: Runway Gen-4.5
- Aspect ratio: 16:9
- Resolution: 1920x1080 minimum (use 4k if credits allow)
- Clip duration targets: Scene 1 = 5s, Scene 2 = 6s, Scene 3 = 7s, Scene 4 = 6s, Scene 5 = 6s
- Motion strength: medium (raise to medium-high only if shot feels too static)
- Camera motion: cinematic, smooth, intentional
- Visual style: cinematic animated realism, family-friendly, clean lighting
- Character consistency: reuse one reference frame from Scene 2 for Scenes 3-5
- Generation strategy: generate 2 takes per scene, select best performance, then run one polished final take

Global negative prompt for all scenes:
blurry faces, extra limbs, distorted hands, horror visuals, weapons, military combat, gore, text artifacts, watermark, logo, low-resolution, jittery camera, chaotic flicker

## Creative Direction

Core hook: ordinary kids are invited into an extraordinary mission.  
Emotional arc: curiosity -> training -> challenge -> courage -> call to action.

## Timeline

### 0:00-0:05
Visual: Dawn light over a quiet town. A mission envelope stamped "CONFIDENTIAL" slides under a front door. A child opens it and their eyes widen.  
On-screen text: RECRUITMENT NOTICE  
Audio SFX: Soft whoosh, paper slide, low pulse beginning.  
Voiceover (vo_01_mission_starts.mp3): "This is a message for new recruits. Your mission starts now."

Runway Gen-4.5 prompt (copy/paste):
Animated cinematic sunrise in a suburban street. A CONFIDENTIAL envelope slides under a front door. Child picks it up, opens it, eyes widen. Warm golden light, shallow depth of field, smooth dolly-in, hopeful mood, family-friendly, 16:9.

Runway notes:
- Keep this scene emotionally warm and curious.
- Avoid fast cuts here. Let it breathe for the first 2 seconds.
- If text generation is unstable, add on-screen text later in edit.

Gen-4.5 fallback prompt:
Sunrise suburban scene, confidential envelope under door, child opens it with wonder, warm cinematic light, smooth dolly-in, family-friendly.

### 0:11-0:18
Visual: Training sequence with age-appropriate obstacles: laser-line crawl, code lock puzzle, map route challenge. Kids help one another instead of racing alone.  
On-screen text: TRAIN. TRUST. MOVE TOGETHER.  
Audio SFX: Footsteps, gadget beeps, rhythmic impacts.

Runway Gen-4.5 prompt (copy/paste):
Kid-safe spy training animation: children crawl under red light lines, solve a glowing code lock, and plan routes on a mission map while helping each other. Teamwork over competition, dynamic side-tracking camera, cinematic depth, no weapons, family-friendly, 16:9.

Runway notes:

Gen-4.5 fallback prompt:
Kid-safe spy training with laser-line crawl, code puzzle, map planning, teamwork focus, dynamic camera, no weapons, cinematic family-friendly action.

### 0:18-0:24
Visual: Command screen flickers with a breach warning. The Null symbol appears, then kids regroup around a strategy board.  
On-screen text: SYSTEM BREACH: THE NULL  
Audio SFX: Short glitch burst, warning ping, bass rise.  
Voiceover (vo_04_stand_together.mp3): "The Null is trying to break the signal. Real agents stand together."

Runway Gen-4.5 prompt (copy/paste):
High-tech command screen flickers with SYSTEM BREACH and The Null symbol. Brief red glitch overlay, then kids regroup around a mission board with determined teamwork. Suspenseful but not scary, controlled camera push-in, cinematic contrast, animated realism, 16:9.

Runway notes:
- Keep glitch moment short, around 1 second visual emphasis.
- Shift quickly from tension to unity and resolve.
- Do not make the symbol horror-like.

Gen-4.5 fallback prompt:
Command screen breach alert with brief glitch, then kids regroup around mission board, determined teamwork, suspenseful but not scary.

### 0:24-0:30
Visual: Heroic final reveal: kids run forward through opening light beams into the event title card.  
On-screen text: MISSION IMPOSSIBLE: VBS 2026  
Subtext: DO YOU ACCEPT THE MISSION?  
Audio SFX: Impact hit, rising cymbal, clean end sting.  
Voiceover (vo_05_do_you_accept.mp3): "Mission Impossible: VBS 2026. Do you accept?"

Runway Gen-4.5 prompt (copy/paste):
Epic family-friendly finale: kids run forward through bright mission light beams, then hold on a bold cinematic end frame for title overlay. Heroic framing, clean light rays, subtle lens flare, triumphant energy, animated realism, 16:9.

Runway notes:
- Generate a clean visual end frame that can hold for title overlay.
- Prefer less character motion in the final 2 seconds to support readable text.
- Add event text in post if Runway text is inconsistent.

Gen-4.5 fallback prompt:

## Edit Notes

- Keep pacing inspirational, not aggressive.
- Favor warm highlights and hopeful expressions in every scene.
- Prioritize clear readability of title text in final 3 seconds.
- Keep dialogue above music by 6-9 dB.

## Runway Export Checklist

- Export each scene as a separate mp4 using the suggested filenames.
- Confirm total sequence target is close to 30 seconds before final assembly.
- If a scene is too short, regenerate with same prompt and extend by 1-2 seconds.
- If character faces drift too much, use image reference frames from Scene 2 for Scenes 3-5.

## Runway Gen-4.5 Workflow

- Pass 1 (blocking): Run all five scenes at 1080p with Gen-4.5 and pick best camera/action take.
- Pass 2 (continuity): Re-run only scenes with character drift using the same reference frame and wording.
- Pass 3 (final): Export locked takes at highest practical quality and keep durations fixed for edit sync.
- Keep prompt phrasing consistent across scenes to improve visual continuity.
