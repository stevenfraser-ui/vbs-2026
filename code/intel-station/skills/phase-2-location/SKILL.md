# Phase 2: Where is The Light?

The agent has discovered that The Light is software created by LOGOS. Now they need to find WHERE it's stored.

## INVESTIGATION FLOW

### Stage 1: Location Search
When the agent asks where The Light is stored:
- Since it's software, it must be on a very powerful computer somewhere
- Initiate an "energy signature scan" — software this powerful needs massive electricity
- Show the world map scan results in the Data Viewer

### Stage 2: Narrowing Down
The energy scan found a massive power draw from one location:
- Show satellite imagery of the building complex
- The facility uses more power than a small city
- Whatever is inside needs enormous computing power to run

### Stage 3: Server Vault Discovery
When the agent wants to look inside:
- Decrypt the building schematic / blueprint
- Point out the "SERVER VAULT" in the center
- Explain: a server vault is like a massive, heavily armored library for computers instead of books
- The Light software is stored on servers deep inside this vault
- Include [ADVANCE] when the agent understands the location
- Prompt: "We know WHAT it is and WHERE it is. Now we need to know HOW it's protected."

## RULES
1. Keep building on the discovery from Phase 1 — reference LOGOS and software
2. Include [ADVANCE] when the agent demonstrates understanding of the server vault location
3. Use spy/mission language throughout
