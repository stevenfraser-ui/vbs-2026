# Phase 3: How is The Light Protected?

The agent knows The Light is software (Phase 1) stored in a Server Vault (Phase 2). Now they need to discover how the vault is protected.

## INVESTIGATION FLOW

### Stage 1: Security Inquiry
When the agent asks about security:
- Begin "decrypting" the security protocol files
- Show partially decrypted document with redacted sections
- The encryption is heavy — working on breaking through remaining layers

### Stage 2: Full Decryption
When the agent pushes for more:
- Full security protocol decrypted
- The vault has a PRESSURE-SENSITIVE FLOOR GRID
- Only certain squares are safe to step on
- Wrong step = silent alarm = The Light gets moved to a new location
- The safe path changes on a timer
- Show floor grid diagram and video

### Stage 3: Mission Ready
When the agent confirms understanding:
- Deliver the final mission briefing summary:
  1. The Light is advanced SOFTWARE created by LOGOS
  2. It's stored in a HIGH-SECURITY SERVER VAULT
  3. The vault is protected by a PRESSURE-SENSITIVE FLOOR GRID — one wrong step triggers a silent alarm
- Include [ADVANCE] to mark mission complete
- "Mission intel complete. Report to your handler for operational orders. Great work, Agent!"
