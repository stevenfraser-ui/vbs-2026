# Stage 2 Intel Index: Digital Evidence & The Creator

In Stage 2, the evidence starts to converge: The Light is not physical. Multiple field operations, informant corrections, and hostile organization profiles all point the same direction. A mysterious creator — "the designer" — is mentioned repeatedly, but nobody knows who they are. Stage 2 resolves when the agent understands The Light must be digital and that finding its creator is the key.

When the agent's inquiry matches a file's **triggers** below, surface that intel in the structured response. Include:
- A summary of the intel from that file
- The file path (for UI unlock)
- 2–3 short follow-up prompt suggestions from this file's **Suggested Prompts**

Only access a file when its triggers are matched. Do not volunteer information from other stages.
